# 2.6 引导上下文

bootstrap.yml 就是引导上下文的配置文件。对于应用，就是来连接Nacos以读取Nacos中的配置。

远程配置(Nacos) & 本地配置优先级

默认情况下,远程配置优先级较高哦

如下的配置文件必须放在远程配置中才生效

spring:
  cloud:
    nacos:
      config:
        # 是否允许本地覆盖远程配置
        allow-override: true
        # 是否一切以本地配置为准，默认false
        override-none: false
        # 系统环境变量或系统属性才能覆盖远程文件的配置
        # 本地配置文件中配置优先级低于远程配置，默认 true
        override-system-properties: true
